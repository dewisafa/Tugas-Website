# 2. TUJUAN PROJEK

Tujuan dari proyek ini adalah:
- Membuat dan mengonfigurasi server website berbasis Nginx.
- Menggunakan GitHub sebagai repositori untuk menyimpan dan mengelola kode website.
- Melakukan proses deploy, uji coba, serta troubleshooting terhadap server.
- Mempelajari kelebihan dan kekurangan Nginx sebagai web server.
- Mendokumentasikan seluruh proses pembuatan server dan solusi terhadap masalah yang muncul.
